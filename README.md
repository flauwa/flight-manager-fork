# Quản lý chuyến bay
Đồ án Nhập môn Công nghệ phần mềm - Nhóm 13.

## Cài đặt
### Install required packages
```
pip install -r requirements.txt
```

### Database migration
```
cd FlightManager
python manage.py migrate
```

### Chạy server
Browse vào thư mục FlightManager,
```
python runserver.py runserver
```
